WeaveRx v0.1.3 is the public launch release — polished docs, hardened core paths, and PyPI-ready install.

**PyPI:** `pip install weaverx`

## Highlights

- **Launch-ready README** — compact hero, centered badges (PyPI, Python, MIT, CI, Docs), pointer-first installation
- **Installation guide** — [fratresmedai.github.io/WeaveRx/installation/](https://fratresmedai.github.io/WeaveRx/installation/) with virtualenv setup and troubleshooting
- **Smarter triage drafts** — few-shot LLM examples + category-specific refinement (repro, dataset access, clinical validation, privacy, duplicates)
- **Resilient GitHub client** — retries on rate limits, typed errors (403/404/429), pagination for comments and recent issues
- **Safety unchanged** — dry-run by default, `--confirm` for writes, local safeguard heuristics

## Quick start

```bash
pip install weaverx
weaverx triage --repo Project-MONAI/MONAI --issue 42 --mock
```

Docs: https://fratresmedai.github.io/WeaveRx/
