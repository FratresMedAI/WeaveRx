WeaveRx v0.1.2 is the launch polish release — PyPI install, clearer docs, and safer defaults for maintainers triaging medical AI issues.

## Highlights

- **PyPI package now live:** `pip install weaverx`
- Major README and documentation improvements
- Added professional badges, governance files, and docs site
- Strengthened safety defaults and responsible use guidance
- Better installation experience and clearer quickstart
- GitHub Action install path fixed for external repos using `FratresMedAI/WeaveRx@v*`

## Install

```bash
pip install weaverx
weaverx triage --repo Project-MONAI/MONAI --issue 42 --mock
```

Docs: https://fratresmedai.github.io/WeaveRx/

## What's next

Embedding-based duplicate detection, PR triage mode, and expanded medical-AI prompt presets are on the roadmap — see [CHANGELOG.md](https://github.com/FratresMedAI/WeaveRx/blob/master/CHANGELOG.md).

Thank you to everyone who tried early builds and filed feedback. Issues and suggestions are welcome on GitHub.
