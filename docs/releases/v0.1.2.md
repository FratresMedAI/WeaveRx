WeaveRx v0.1.2 is the launch polish release.

**PyPI is now live** — `pip install weaverx`

This release focuses on making WeaveRx easier to adopt for medical AI maintainers:

- Dedicated [installation guide](https://fratresmedai.github.io/WeaveRx/installation/) with virtual environment recommendations and troubleshooting
- Cleaner README with improved badge layout and a short, pointer-first Installation section
- Professional shields.io badges and better first-impression structure
- Strengthened safety defaults and responsible use messaging throughout
- GitHub Action install path fixed so `uses: FratresMedAI/WeaveRx@v*` works cleanly from external repos
- Full governance docs (ETHICS.md, SECURITY.md, etc.) and MkDocs site now live

**Quick start**

```bash
pip install weaverx
weaverx triage --repo Project-MONAI/MONAI --issue 42 --mock
```

See the [full documentation](https://fratresmedai.github.io/WeaveRx/) and [CHANGELOG](https://github.com/FratresMedAI/WeaveRx/blob/master/CHANGELOG.md).

Roadmap items (embedding duplicate detection, PR triage mode, expanded medical-AI presets) remain planned for future releases.

Thanks to everyone who gave early feedback.
